# 🤖 AI-Agents-Config

集中管理 AI Agent 的全域規範、專案規範、Prompt、Design System 與開發工作流。

本 Repository 作為 AI Agent 的知識庫（Knowledge Base），提供可重複使用的設定與規範，協助不同專案維持一致的開發流程與輸出品質。

---

# 🎯 Goals

- 建立一致的 AI 開發流程
- 減少重複撰寫 Prompt
- 提高 AI 輸出品質與一致性
- 建立可維護、可擴充的 Agent 知識庫
- 讓不同專案共享通用規範

---

# 📁 Repository Structure

```text
.
├── global/          # 全域規範
├── projects/        # 各專案專屬規範
├── prompts/         # 常用 Prompt
├── design/          # Design System
├── workflows/       # AI 工作流程
├── templates/       # 可重複使用的模板
└── README.md
```

> 實際目錄會依需求持續調整。
